# Mathematical Operations

### The typical `+ - * /`

* Result from a division will always return as a **floating number**.
* Eg: `print(6/3) # returns 2.0`
* **Priority list:** 
  1. **\(\)**
  2. **\*\***
  3. **\***
  4. **/**
  5. **+**
  6. **-**

```python
print(3 * 3 + 3 / 3 - 3 ) # returns 7.0
```
