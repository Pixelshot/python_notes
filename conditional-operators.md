# Conditional Operators

### Syntax

```python
if condition:
    do this
else:
    do this
    
#Eg
water_level = 50
if water_level > 80:
    print("Drain water")
else:
    print("Continue")
```

#### Nested Syntax

```python
if condition1:
    if condition2:
    do something
    else:
    do something
else:
    do something
```

#### if / elif / else

```python
if condition1:
    do A
elif condition2:
    do B
else:
    do this
```
